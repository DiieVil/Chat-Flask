El fichero incluye en proyecto final de Tecnologías Web del grupo formado por Juan Terrer, Marc Usó, Jaume Martinis y Diego Vicente.

Se trata de un chat multicanal contruido alrededor de la tecnología backend Flask, acompañado de Javascript, HTML y CSS.

En este se pueden encontrar 2 carpetas diferentes. Una llamada static, donde se encuentran los ficheros javascript, 
junto a la imágen de la upv necesaria para que se muestre en la página web correctamente.
En los ficheros javascript, se encuentran los documentos chatroom.js y chatlist.js los cuáles se encargan de comunicar y 
retransmitir información a los archivos python y html.

En la carpeta templates encontramos todos los archivos HTML que conforman la página web (base.html, error.html, 
index.html, chatlist.html y chatroom.html)

Después encontramos el archivo application.py, que es el archivo python que ejecuta y controla toda la página web. En esta
se crean todas las variables globales, de sesión y eventos los cuáles permiten el buen funcionamiento de la web.

Por último, encontramos el archivo de texto requirements, los cuáles indica los distintos archivos que se necesitan para
la correcta ejecución de la página web.
